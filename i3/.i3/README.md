# Dependencies

 * *Fira Code* default font
 * *ttf-font-awesome* font for icons
 * *rofi* application launcher
 * *i3blocks* status bar
 * *dunst* notifications
 * *nm-applet* network manager
 * *feh* for wallpaper
 * *Arc-Dark* gtk theme

## Status bar dependencies
 * *xkb-switch* for keyboard layout
 * *acpi*
 * *sysstat*
 * *xdotool*
 * *xbacklight* for screan brightness
